    많이 쓰이는 기본 동작

    1. element로 마우스 이동: ActionChains(driver).move_to_element(ref)
    2. element로 마우스 클릭: ActionChains(driver).click(ref)
    3. element 키보드 입력 : ActionChains(driver).send_keys_to_element(ref,keys)
    4. 키보드 입력: ActionChains(driver).send_keys(keys)
  • 위 네 가지 사항 중, 현재 프로젝트 내 한경컨센서스 제목 크롤링에서는 마우스 이동하면 뜨는 것을 수집하는 것이 필요하기 때문에 첫 번째 코드를 사용했다.

    enumerate 함수

  • text.split('\n'): 공백 (스페이스, 엔터, 탭 등) 기준으로 문자열을 나눈다 (출처: 위키독스 점프 투 파이썬) - enumerate(rows): "열거하다" 라는 뜻. 순서가 있는 자료형 (리스트, 튜플, 문자열) 입력받아 인덱스 값을 포함하는 enumerate 객체로 돌려줌
  • for _, row in ~~: 이런 코드를 사용하셨길래 _, 의 의미를 여쭤봤다! 아직 많은 코드를 보지는 못 했지만 이렇게 생긴 코드는 무슨 의미일지 검색해봐도 잘 나오지 않아서 여쭤보았다. (코딩 초보이자 물음표 살인마..) -> enumerate로 하면 return 이 2개가 됩니다. 첫번째 argument가 index인데 안쓸거라서 그냥 _ 이런식으로 표현해놓은 거에요 라고 답변 주셨다. enumerate 함수에 대해 조금 더 찾아봐야겠다.
